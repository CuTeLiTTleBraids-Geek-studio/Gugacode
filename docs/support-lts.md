# 支持策略与 LTS 讨论稿（prompt-12 12-N）

> 非正式承诺；用于 1.0 前对齐社区预期。

## 当前（0.x）

| 线 | 支持 |
|---|---|
| `0.4.x` | 主动开发 + 安全修复 |
| `0.3.x` | 关键安全视严重性回移 |
| 更早 | 不支持 |

## 拟议 v1.0

- **LTS 窗口**：1.0.x 发布后至少 **12 个月** 安全补丁。  
- **次要版本**：3–6 个月功能里程碑。  
- **破坏性变更**：仅在 major；0.x 仍可能有边界破坏但 CHANGELOG 标明。  
- **语言矩阵**：Go + TypeScript/JavaScript 为核心；其它语言仅扩展。
